# Português (Brasil)
OBPC Jardim Guaraú – Website Oficial
Este projeto consiste no desenvolvimento do website oficial da Igreja Evangélica O Brasil Para Cristo – Jardim Guaraú.
O objetivo é oferecer uma plataforma moderna, responsiva e acessível, permitindo que membros e visitantes encontrem informações sobre cultos, localização, contato e pedidos de oração.

O desenvolvimento utilizou práticas atuais de front‑end e contou com apoio de IA Generativa Assistida.

Tecnologias Utilizadas
HTML5

CSS3

JavaScript

Responsividade com Flexbox e Media Queries

Boas práticas de acessibilidade

Estrutura preparada para integração futura com backend Node.js

Funcionalidades
Menu responsivo para dispositivos móveis

Seção institucional sobre a igreja

Grade de cultos

Formulário de pedido de oração com validação e limitação de envio

Mapa integrado via Google Maps

Arquitetura preparada para expansão futura
